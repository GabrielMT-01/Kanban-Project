Kanban Board

Modern Kanban board built with React, TypeScript, Tailwind CSS, and DnD Kit.
Designed with focus on responsive UI, smooth interactions, accessibility, and real-world frontend architecture.

        Features
Drag and drop tasks between columns
Responsive layout for mobile, tablet, and desktop
Dark / Light mode
English / Portuguese language support
Search tasks by title
Column sorting and filtering
Inline task editing
Comments system
Task priorities
Persistent data with LocalStorage
Transitions
